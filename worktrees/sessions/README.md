# Worktree Sessions

Each worktree push session gets a file here, created by the master agent during planning.

**Naming:** `YYYY-MM-DD-<slug>.md` (e.g., `2026-04-01-sprint13-push.md`)

**Template:** `_template.md`

The session file tracks:
- Which lanes are active
- Branch names and worktree assignments
- Merge order and status
- Running notes from the master
