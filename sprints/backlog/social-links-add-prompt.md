# Social Links — Prompt to Add More

## Problem
Profile page shows existing social links (e.g. Instagram) but doesn't prompt the user to add more platforms. Users may not know they can add LinkedIn, TikTok, etc.

## Proposed Solution
Below the existing social links on the profile settings page, show an "Add social link" button that reveals a dropdown/picker for common platforms:
- Instagram
- LinkedIn
- TikTok
- Facebook
- Twitter/X
- YouTube
- Personal website

Each platform shows an icon + placeholder URL field. Already-added platforms are hidden from the picker.

## Notes
- Social links display in the hero section of the public profile
- Keep it simple — icon + URL is enough, no validation beyond URL format
