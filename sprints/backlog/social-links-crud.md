# Social Links Add/Remove from Profile

## Problem
The social links row on My Profile shows existing links (Instagram, LinkedIn, Website) as icons, but there's no way to add or remove social links directly from there. Users have to navigate away to settings.

## Vision
Inline add/remove for social links on the Profile page. The social links bar should have an affordance to:
- Add a new social link (from a set of supported platforms)
- Remove an existing link
- Edit an existing link URL

## Notes
- Keep it discreet — the bar looks clean now, don't clutter it
- Could use a small "+" icon at the end of the row, and long-press/tap to edit/remove
- Or tap the row to enter an edit mode showing all links with add/remove controls
