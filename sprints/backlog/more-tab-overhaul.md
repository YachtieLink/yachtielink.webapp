# More Tab — The Net That Catches Everything Else

**Status:** idea
**Priority guess:** P2 (organizational, not a growth driver — but needs to not be a dumping ground)
**Date captured:** 2026-04-02

## Problem
The More tab is currently "Settings" — a flat list of account management items. As the product grows, it needs to absorb things that don't belong on the 4 main tabs but still need a home. Without intentional design, it becomes a junk drawer.

## What moves INTO More (from other tabs)
- **Cert Document Manager** — from Insights (doesn't belong there). Link to it from Certifications on Profile.
- **Crew Pro subscription management** — from Insights. This is account management.
- **Profile Settings** (visibility toggles, view mode) — currently on its own page, should be accessible from More

## What's already in More
- Appearance (dark mode placeholder)
- Edit profile & contact info
- Saved Profiles
- Login & security
- Download my data (GDPR)
- Delete my account
- Crew Pro billing
- Feature Roadmap
- Send feedback
- Terms of Service
- Privacy Policy
- Sign out

## Notes
- Sand section color (already applied)
- Keep it organized and honest — this IS the utility drawer, but it should be a TIDY utility drawer
- As features grow (messaging, notifications, privacy controls, blocked users, data exports, API keys for agents), they'll land here first. Plan the IA for growth.
- Don't overthink this — it's the least exciting tab and that's fine. Just don't let it rot.
