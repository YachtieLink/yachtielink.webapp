# App Tab Section Flow — Match Public Profile Quality

**Status:** idea
**Priority guess:** P2 (important — visual polish for launch quality)
**Date captured:** 2026-04-01

## Summary
The public profile page has a clear visual hierarchy: hero → stat summary → distinct sections (About Me, My Experience, My Interests) each with icon + heading + breathing room. The main app tabs (Profile, Network, Insights) feel flat and dense by comparison. Apply the same organized, flowing section pattern to the core tabs.

## What "good" looks like (public profile)
- Each section has an icon + label header
- Clear vertical spacing between sections
- Content flows naturally top-to-bottom
- Sections feel distinct but cohesive

## What needs work
- **Profile tab:** Dense settings-grid of toggles. Sections exist but feel cramped. The toggle grid is functional but doesn't invite exploration.
- **Network tab:** Flat list with tab switcher. Colleague cards are good but the page lacks section rhythm.
- **Insights tab:** Sparse — just stats and cards with lots of empty space below.

## Notes
- This is a layout/IA pass, not a feature build
- The inner-page-header component (Lane 1, current session) is a prerequisite — get consistent headers first, then address section flow
- The public profile is the north star for visual quality
- Founder feedback: "see how this page looks? I'd like to capture that over the other tabs as well, it's organized and has flow to it"
