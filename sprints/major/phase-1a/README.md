## Phase Goal

Photo-forward profiles, Notion-level visual energy, collapsible sections with smart summaries, save/bookmark functionality, hidden-by-default empty sections. The "Instant Good Profile" — a CV parse should leave you with something that looks polished immediately, not a skeleton.

Design philosophy: mobile-first, photo-forward, progressive disclosure, empty = invisible, AI-invisible, crew-first trust model.
