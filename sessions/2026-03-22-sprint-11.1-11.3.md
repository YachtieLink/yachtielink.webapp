## Lessons Learned

- `yacht_crew` table doesn't exist — the actual table is `attachments`. Subagents hallucinate table names based on function names in the codebase.
- `certifications` has no `sort_order` — use `created_at` for ordering.
- Switching from native `<Select>` to custom component loses the empty option — always add a clearable mechanism.
- Fail-open defaults (isPro = false on error) are worse than fail-closed for paid features — default to showing more, server enforces limits.
